Enunciado

Un grafo orientado es un digrafo D tal que al menos uno de \(v \rightarrow w\) y \(w \rightarrow v\) no es una arista de \(D\), para todo \(v,w \in V(D)\). En otras palabras, un grafo orientado se obtiene a partir de un grafo no orientado dando una dirección a cada arista. Demostrar en forma constructiva que para cada \(n\) existe un único grafo orientado cuyos vértices tienen todos grados de salida distintos.

Ayuda: aprovechar el ejercicio anterior y observar que el absurdo no se produce para un único grafo orientado.

Solución

Definimos grafo orientado como \(D~|~v \rightarrow w \in E(D) \oplus w \rightarrow v \in E(D)\)

Y quiero probar que \[\forall G = (V,E),|V| = n,|E| = m:\exists!D = \overset{\rightarrow}{G}::\nexists u,v \in V~|~d_{\text{out }}(u) = d_{\text{out }}(v)\]

Esto se puede demostrar por construcción.

Queremos armar \(\overset{\rightarrow}{G}\) a partir de \(G\) tal que todos sus nodos tengan grado de salida distinto y ver que solo esa solución es posible.

Entonces tenemos \(V = \left\{ v_{1},\ldots v_{n} \right\}\) y le damos orientación tal que no hay aristas bidireccionales

\[\begin{array}{r} d_{\text{out }}\left( v_{1} \right) = 0, \\ d_{\text{out }}\left( v_{2} \right) = 1, \\ \ldots, \\ d_{\text{out }}\left( v_{n - 1} \right) = n - 2, \\ d_{\text{out }}\left( v_{n} \right) = n - 1 \end{array}\]

Ver que no existe otra asignación es trivial, cada nodo puede estar conectado a \(n - 1\) nodos máximo y \(0\) nodos mínimo.

Nota: este etiquetado es arbitrario, esto vale para cualquier \(G'\) isomorfo a \(G\)

Del proceso anterior tenemos también que:

\[\sum_{i = 1}^{n}d_{\text{out }}\left( v_{i} \right) = \sum_{i = 0}^{n - 1}i = \frac{(n - 1)(n - 1 + 1)}{2} = \frac{n(n - 1)}{2}\]

Esto nos indica que es un grafo completo, o sea, de \(\frac{n(n - 1)}{2}\) aristas (en este caso dirigidas, pero vale ya que no hay bidireccionales).

Esto nos dice que existe un único \(D = \overset{\rightarrow}{G}\) tal que todos los vértices tienen grado de salida distintos, y en particular, es \(K_{n}\). \(😀\)
